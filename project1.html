Built Bristo Web and Mobile Restaurant Application

  • Live Demo
  • Source code

Bristo application is an app that enables Bristo restaurant to sell their customers orders for food items online, all they have to do is open the Bristo mobile application or website to access the food items and then add them to the cart

It has both web and mobile versions and the mobile version runs cross-platform for both android and IOS

  • The web version of the application is built using React,react-router-dom, tailwindcss, and styled-components
  • The mobile version is built using React-Native,react-navigation, and tailwindcss
  • For state management, I used redux and redux persist to maintain state even after mobile application is closed
