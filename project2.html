Built Notes Mobile Application

  • Live Demo
  • Source code

Notes Mobile application is an application that enables authenticated users to create notes of which the notes are stored in the database and whenever the user signs in, his/her notes are retrieved.

This is a full-stack mobile application built using the MERN(MongoDB, Express, React Node.js) Stack in which the users can create, delete, update and read their notes.

  • The Mobile version is built using React-Native, tailwindccs and react-navigation
  • The backend of this application is built using Node.js, Express and MongoDB
  • For state management, I used Redux and Redux-perist
